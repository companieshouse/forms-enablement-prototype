Additional documents

You need to file other documents as well as the Statement of Capital before the company’s capital can be reduced. If the relevant documents have already been filed, there is no need to file them again. The Statement of Capital will not be processed or entered onto the company’s register until all the relevant documents have been filed successfully.
You can upload additional documents with this application or the Statement of Capital can be submitted by itself.

Documents you can also submit

If you wish to use the same day service, we will need all these documents. If you have already sent one of these documents, you do not need to send it again.

If the company has made other changes to its capital, you will need to file any other capital forms separately.
